# 🚀 DigiTools

**DigiTools** is a modern and responsive SaaS landing page built with **React**. It showcases premium digital tools, workflow presentation, pricing plans, ratings, and a clean call-to-action based layout.

This project demonstrates reusable components, structured folder organization, and responsive UI design using modern frontend technologies.

---

## 🌐 Live Demo

👉 https://mezba-digi-tools-a06.netlify.app/

---

## 🌟 Features

- 🚀 Modern Hero Section with strong CTA  
- 💎 Premium Digital Tools showcase  
- ⭐ Rating statistics display  
- 🔄 Workflow explanation section  
- 🧭 3-Step Getting Started guide  
- 💰 Pricing Plans (Basic / Pro / Enterprise)  
- 🎨 Gradient-based modern UI  
- 📱 Fully responsive design  

---

## 💻 Tech Stack

- **Frontend:** React.js, Tailwind CSS, DaisyUI  
- **Language:** JavaScript (ES6+)  
- **Libraries:** React-Toastify  
- **Data:** JSON  
- **Build Tool:** Vite
